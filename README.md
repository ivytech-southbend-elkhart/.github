# WELCOME

This organisation and the associated repositories are for academic software developement, learning, and collaboration between students and instructors of [Ivy Tech Community College](https://www.ivytech.edu/) in South Bend and Elkhart, Indiana.

## REPOSITORY DISPOSITION

Repositories with student information **must be private**--e.g. classroom repositories where students submit assignments. Assignment template repositories *may* be public if they **do not** contain proprietary or copyrighted information that prevents public release.

#### NOTICE

Neither this GitHub organization nor any of the repositories within it are owned, operated, or sponsored by Ivy Tech Community College, any government agencies, or any for-profit or not-for-profit entities. They are strictly a grass-roots effort by both students and instructors to facilitate learning.
